# Gatsby plugin for Criteo
Simple gatsby plugin that adds Criteo script to \<head>.
## Install
```
npm i -D gatsby-plugin-criteo
```

## Usage
```
[
  `gatsby-plugin-criteo`,
  // or
  {
    resolve: `gatsby-plugin-criteo`,
    options: {
      includeInDevelopment: false // default false
    }
  }
]
```
